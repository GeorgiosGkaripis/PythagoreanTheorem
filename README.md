## 📝 Περιγραφή
Ένα εξειδικευμένο γεωμετρικό εργαλείο για τον άμεσο υπολογισμό της υποτείνουσας ή των κάθετων πλευρών ενός ορθογωνίου τριγώνου. Αναπτύχθηκε για να προσφέρει ταχύτητα και ακρίβεια σε μελέτες πεδίου, αρχιτεκτονικές χαράξεις και τεχνικούς ελέγχους.

## 🚀 Κύριες Λειτουργίες
* **Επίλυση Ορθογωνίου Τριγώνου:** Υπολογισμός της τρίτης πλευράς (α, β ή γ) με βάση το θεώρημα $a^2 + b^2 = c^2$.
* **Τεχνική Ακρίβεια:** Ιδανικό για τον έλεγχο "γωνιάσματος" οικοδομικών στοιχείων και μεταλλικών κατασκευών.
* **Responsive Interface:** Βελτιστοποιημένο για χρήση από smartphone, διευκολύνοντας τον μηχανικό/σχεδιαστή απευθείας στο εργοτάξιο.
* **Code Protection:** Χρήση τεχνικών obfuscation για την προστασία του πηγαίου κώδικα.
